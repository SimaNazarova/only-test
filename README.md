# Лендинг «Исторические даты»‎

[посмотреть лендинг](https://only-test-mk8u.vercel.app/)


#### установка проекта
```
 npm install
```

#### запуск проекта
```
 npm run dev
```


#### В проекте использованы:
1. Next.js
2. TypeScript
3. Стейт менеджер Zustand
4. SCSS
5. GSAP
6. Swiper
7. React-hook-form + yup
